- **Inbound:** Job submission (AGI experiments, game servers)
- **Outbound:** Job status, results, capsules
- **Determinism:** Guaranteed for same seed/config
- **Isolation:** Network-isolated tenants
- **Monitoring:** Real-time metrics via `/metrics` endpoint

### 3. Autocoder â†” Product Repos
**Interface:** Git + GitHub API
- **Inbound:** Product type, configuration parameters
- **Outbound:** Patch archives, Draft PRs
- **Idempotency:** Multiple runs produce same output
- **Audit:** All operations logged
- **Rollback:** Patches can be reverted

**Interface:** Hardware Abstraction Layer
- **Inbound:** Model loading requests, inference jobs
- **Outbound:** GPU metrics, inference results
- **Compatibility:** CUDA, ROCm, Metal
- **Simulation:** Can mock GPUs for CI

### 5. Demo Build â†” Marketing/Sales
**Interface:** Artifact Repository
- **Inbound:** Demo scenario requests
- **Outbound:** Compiled demo artifacts, screenshots, videos
- **Versioning:** Tied to product releases
- **Security:** No PII, no production secrets
